    Meus Projetos

    Listados abaixo estão os projetos que realizei

    Projeto de Xadrez

    Nesse projeto eu desenvolvi um jogo de xadrez em java, jogavel a partir de um console interativo e colorido (git bash)

    Jogo Pong

    Esse projeto eu recriei o jogo Pong com HTML, CSS e JavaScript. O projeto foi desenvolvido em orientação a objetos e pode ser jogavel a partir de um navegador.

    Labirinto IA

    Neste projeto eu desenvolvi um labirinto 10x10, onde uma inteligencia artificial utiliza do algorítmo BFS para navegar entre o labirinto e fazer sempre uso do caminho mais curto para chegar a saida

    Sistema de Voucher para Pousadas

    Feito em NodeJS e JavaScript, realizei um projeto onde um usuario poderá cadastrar clientes e suas informações de hospedagem para armazenar em um banco de dados (MongoDB) e fazer o download de um PDF do voucher.

    Sistema de Biblioteca

    Nesse projeto, eu desenvolvi um sistema completo de gerenciamento de bibliotecas onde o administrador pode cadastrar livros, autores, emprestimos e resgatar relatórios sobre valores devidos e livros que foram imprestados.

    Site de Desenvolvimento Web

    Um site realizado em parceria com dois amigos que desenvolvemos, o site é um portifólio para uma empresa e nele contem informações sobre serviços, apresentações e mais...

    Sistema de Gestão de Universidade

    Nesse sistema, eu desenvolvi em colaboração com dois amigos um gerenciamento de uma universidade, onde um coordenador, pode gerenciar estudantes, professores, matriculas, cursos e disciplinas.

    API Park

    Esse foi o desenvolvimento de uma API de gerenciamento de vagas e usuarios para um estacionamento, onde o administrador do sistema pode marcar vagas livres e ocupadas, o sistema utiliza java e Spring com diversas dependencias, tais como JWT, Swagger, JUnit e mais...

    Microsserviços para Previsão do Tempo

    Quando eu estava estudando sobre serviços de Mensageria, eu desenvolvi esse com arquitetura em microsserviços, o sistema recebe um uma cidade do producer e recebido no consumer, ao receber a informação, o Consumer processa a informação recebida utilizando uma integração com uma API de previsão do tempo.

    Sistema de Pedidos com Deploy na AWS

    Esse sistema utiliza de rotas HTTP para fazer requisição entre 3 microsserviços: ms-user, ms-stock e ms-order, nesse sistema, é realizado deploy na AWS, com video apresentativo e logs de informações/erros. O sistema conta com documentação, logs de erro, Hateoas de navegação e outras tecnologias.
